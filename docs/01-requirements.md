# 要件定義

## 目的
良かったYouTube動画を後から見返せる公開コレクションを作る。

## スコープ
- 一般公開のリスト/詳細ページ
- 管理者(単一ユーザー)が作成/編集/削除
- 管理者向けにログイン/追加/編集画面を提供

## データ項目(1件あたり)
- YouTube URL(必須)
- タイトル(保存時に自動取得)
- サムネURL(保存時に自動取得)
- タグ(自由入力、各10文字以内)
- カテゴリ(自由入力、10文字以内)
- 良かったポイント(markdown、2000文字以内)
- メモ(markdown、2000文字以内)
- 良かったレベル(1-5)
- 公開日(将来拡張用、現状は未使用)
- 追加日(created_at)

## リストの並び順
- 追加日(新しい順)
- 公開日(新しい順)[将来拡張]
- 良かったレベル(高い順)

## Markdown表示
- HTMLは無効
- 出力はエスケープ
- 画像埋め込みは許可

## 決定事項
- タイトル/サムネは保存時に取得して保存
- 公開範囲は完全公開
- 公開日は将来拡張として保留
- 技術スタックは Next.js/TypeScript + Supabase Auth + Prisma
- 本番デプロイは Vercel（`main` ブランチ、`front/` のみ）
- フロント実装は `front/` フォルダに新規作成
- データは Supabase Postgres を前提に実装（インメモリは廃止）
