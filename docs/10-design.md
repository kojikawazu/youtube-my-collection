# デザイン設計

## 全体方針
- YouTubeで動画を探すときの「見やすさ・探しやすさ」を優先
- サムネ中心のカードUIで視認性を重視
- 薄赤のライトテーマで目に優しい体験
- モバイル/デスクトップ両対応

## ビジュアルトーン
- 色味: 淡い赤/ピンク + 柔らかいニュートラル
- コントラスト: 強すぎないが視認性は確保
- 形状: 角丸、柔らかい境界線、控えめな影
- 背景: 単色ではなくごく薄い赤系のトーン

## タイポグラフィ
- 情報の階層を明確にするサイズ差
- タイトルは太め、補助情報は小さく軽め
- 可読性重視、行間と余白は広め

## 画面別設計

### リスト画面
- サムネ中心のカードグリッド
- カード内に: タイトル / タグ / カテゴリ / 良かったレベル
- 並び替えUIは目立ちすぎないユーティリティ位置
- 管理者操作は小さなボタンで控えめに表示

### 詳細画面
- サムネ + タイトルを強調
- タグ / カテゴリ / レベルは補助情報として整理
- 良かったポイント / メモはMarkdown表示
- YouTubeリンクは明確にクリック可能

### ログイン画面
- 「Googleでサインイン」ボタンのみ
- 管理者専用である旨の短い補足
- 軽量でミニマルな構成

## UIパーツ
- ボタン: 小さめ、角丸、薄い境界線
- 入力: 柔らかい枠線、フォーカス時に淡いハイライト
- タグ: 小さなチップ表示
- 評価: 1-5の星やバーで直感表示

## アニメーション
- 軽いフェード/スライドで遷移を演出
- サムネホバー時に微妙な拡大
- 速度はゆっくりめで目に優しく
