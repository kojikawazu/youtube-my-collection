## 画面仕様

### リスト画面

- レイアウト: サムネ中心のカード表示
- カード内表示: サムネ画像 / タイトル / タグ / カテゴリ / 良かったレベル(1-5)
- 検索: キーワード検索（タイトル部分一致 + タグ一致）
- 並び替え: 追加日(新しい順) / 公開日(新しい順)[将来拡張] / 良かったレベル(高い順)
- ページング:
  - 1 ページあたり 10 件
  - 「前へ」「次へ」ボタン
  - 現在ページ/総ページ表示（例: `1 / 5`）
  - ページ番号ボタンは最大 5 件表示（例: `3 4 5 6 7`）
  - 一致件数が 0 件のときはページング UI を表示しない
  - 検索・並び替え変更時は 1 ページ目へ戻る
- 管理者操作(許可ユーザーのみ): 追加 / 編集 / 削除
- 管理操作の成功時に右上へ通知を表示(追加/編集/削除)
- バリデーションエラーは入力欄とメッセージで強調表示

### 詳細画面

- 表示項目: タイトル / サムネ画像 / YouTube URL(リンク) / タグ / カテゴリ / 良かったレベル(1-5) / 良かったポイント(markdown) / メモ(markdown)
- 管理者操作(許可ユーザーのみ): 編集 / 削除
- 管理操作の成功時に右上へ通知を表示(追加/編集/削除)
- バリデーションエラーは入力欄とメッセージで強調表示

### ログイン画面

- 管理者専用のログイン導線
- 「Google でログイン」ボタンのみ
- 管理者専用である旨の短い補足

### 追加/編集画面

- 入力項目: YouTube URL / タイトル / タグ / カテゴリ / 評価 / 良かったポイント / メモ
- 必須・上限バリデーションを表示
- 保存成功で右上通知

## 画面遷移

- リスト → 詳細: カードクリック
- 詳細 → リスト: 戻るリンク
- リスト → ログイン: ヘッダーのログインアイコン
- ログイン → リスト: ログイン完了時にリダイレクト
- リスト/詳細 → 追加/編集: 管理者のみ

## UI/UX 方針

### 全体方針

- YouTube で動画を探すときの「見やすさ・探しやすさ」を優先
- サムネ中心のカード UI で視認性を重視
- 薄赤のライトテーマで目に優しい体験
- モバイル/デスクトップ両対応

### ビジュアルトーン

- 色味: 淡い赤/ピンク + 柔らかいニュートラル（背景はごく薄い赤系トーン、`#fff4f4` 系）
- コントラスト: 強すぎないが視認性は確保
- 形状: 角丸、柔らかい境界線、控えめな影

### タイポグラフィ

- 情報の階層を明確にするサイズ差（タイトルは太め、補助情報は小さく軽め）
- 可読性重視、行間と余白は広め

### UI パーツ

- ボタン: 小さめ、角丸、薄い境界線
- 入力: 柔らかい枠線、フォーカス時に淡いハイライト
- タグ: 小さなチップ表示
- 評価: 1-5 の星やバーで直感表示

### アニメーション

- 軽いフェード/スライドで遷移を演出
- サムネホバー時に微妙な拡大
- 速度はゆっくりめで目に優しく

> コンポーネント分割（Atomic Design）の設計は [`notes/atomic-design-plan.md`](./notes/atomic-design-plan.md) を参照。
