# 画面仕様

## リスト画面
- レイアウト: サムネ中心のカード表示
- カード内表示:
  - サムネ画像
  - タイトル
  - タグ
  - カテゴリ
  - 良かったレベル(1-5)
- 検索:
  - キーワード検索（タイトル部分一致 + タグ一致）
- 並び替え:
  - 追加日(新しい順)
  - 公開日(新しい順)[将来拡張]
  - 良かったレベル(高い順)
- ページング:
  - 1ページあたり10件
  - 「前へ」「次へ」ボタン
  - 現在ページ/総ページ表示（例: `1 / 5`）
  - ページ番号ボタンは最大5件表示（例: `3 4 5 6 7`）
  - 一致件数が0件のときはページングUIを表示しない
  - 検索・並び替え変更時は1ページ目へ戻る
- 管理者操作(許可ユーザーのみ):
  - 追加
  - 編集
  - 削除
- 管理操作の成功時に右上へ通知を表示(追加/編集/削除)
- バリデーションエラーは入力欄とメッセージで強調表示

## 詳細画面
- 表示項目:
  - タイトル
  - サムネ画像
  - YouTube URL(リンク)
  - タグ
  - カテゴリ
  - 良かったレベル(1-5)
  - 良かったポイント(markdown)
  - メモ(markdown)
- 管理者操作(許可ユーザーのみ):
  - 編集
  - 削除
- 管理操作の成功時に右上へ通知を表示(追加/編集/削除)
- バリデーションエラーは入力欄とメッセージで強調表示

## ログイン画面
- 管理者専用のログイン導線
- 「Googleでログイン」ボタンのみ
- 管理者専用である旨の短い補足

## 追加/編集画面
- YouTube URL / タイトル / タグ / カテゴリ / 評価 / 良かったポイント / メモ
- 必須・上限バリデーションを表示
- 保存成功で右上通知

## 画面遷移
- リスト -> 詳細: カードクリック
- 詳細 -> リスト: 戻るリンク
- リスト -> ログイン: ヘッダーのログインアイコン
- ログイン -> リスト: ログイン完了時にリダイレクト
- リスト/詳細 -> 追加/編集: 管理者のみ
