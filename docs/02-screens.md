# 画面仕様

## リスト画面
- レイアウト: サムネ中心のカード表示
- カード内表示:
  - サムネ画像
  - タイトル
  - タグ
  - カテゴリ
  - 良かったレベル(1-5)
- 並び替え:
  - 追加日(新しい順)
  - 公開日(新しい順)[将来拡張]
  - 良かったレベル(高い順)
- 管理者操作(許可ユーザーのみ):
  - 追加
  - 編集
  - 削除
- 管理操作の成功時に右上へ通知を表示(追加/編集/削除)

## 詳細画面
- 表示項目:
  - タイトル
  - サムネ画像
  - YouTube URL(リンク)
  - タグ
  - カテゴリ
  - 良かったレベル(1-5)
  - 良かったポイント(markdown)
  - メモ(markdown)
- 管理者操作(許可ユーザーのみ):
  - 編集
  - 削除
- 管理操作の成功時に右上へ通知を表示(追加/編集/削除)

## 画面遷移
- リスト -> 詳細: カードクリック
- 詳細 -> リスト: 戻るリンク
