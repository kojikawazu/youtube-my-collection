# YouTube My Collection (front)

Next.js App Router + TypeScript + Tailwind CSS のフロント実装です。

## Getting Started
```bash
npm install
npm run dev
```

### その他コマンド
```bash
npm run lint
npm run build
npm run start
npm run test:e2e
```

## 設定
- 環境変数は `front/.env.local` に配置
- Supabase / Prisma / 管理者メールは `docs/11-supabase-setup.md` を参照

## ドキュメント
詳細は `docs/` を参照してください。
