# YouTube My Collection (front)

Next.js App Router + TypeScript + Tailwind CSS のフロント実装です。

## Getting Started
```bash
pnpm install
pnpm dev
```

### その他コマンド
```bash
pnpm lint
pnpm build
pnpm start
pnpm test            # ユニットテスト（Vitest）
pnpm test:watch      # ユニットテスト watch モード
pnpm test:e2e        # E2Eテスト（Playwright）
```

## 設定
- 環境変数は `front/.env.local` に配置
- Supabase / Prisma / 管理者メールは `docs/09-architecture-specification.md` を参照
