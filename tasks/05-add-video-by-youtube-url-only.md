# タスク05: YouTube URLのみで動画を追加できるページ

## 目的
管理者が最小入力で動画を登録できるようにし、登録作業の手間を減らす。

## 範囲
- YouTube URL入力のみで登録できる専用ページの追加
- 登録時にタイトル/サムネイルを自動取得してSupabase DBへ保存
- 管理者のみ利用できる導線と権限制御
- バリデーション/エラーハンドリングの実装
- 関連ドキュメント更新

## 要件
- 入力項目は YouTube URL のみ（必須）
- 送信後、`video_entries` にレコードが作成される
- タイトル/サムネイルは既存仕様どおり保存時に自動取得する
- 非管理者または未ログインは利用不可
- 登録成功時は既存UI方針に合わせて通知を表示する
- 登録失敗時は原因が分かるメッセージを表示する

## 想定変更箇所
- `front/src/app/`（新規ページ/導線）
- `front/src/app/api/videos/route.ts`（URLのみ登録時のハンドリング確認）
- `front/src/lib/validation.ts`（URLバリデーション見直しが必要なら）
- `docs/02-screens.md`（画面仕様）
- `docs/06-api.md`（API利用パターン追記が必要なら）

## 手順（案）
- URLのみ登録ページのUI仕様を確定する
- 既存 `POST /api/videos` でURLのみ登録が成立するか確認する
- 必要に応じてAPI側の必須条件/補完処理を調整する
- 管理者導線（表示条件・遷移）を追加する
- 成功/失敗トーストと入力バリデーションを実装する
- E2Eまたは手動確認観点を追加する

## 受け入れ基準
- 管理者がYouTube URLのみで動画を登録できる
- 登録後に一覧/詳細でタイトルとサムネイルが確認できる
- 非管理者からはページ/操作にアクセスできない
- 既存の動画追加フローを壊さない
